A weather website.
using:
  PHP, javascript, html, css, jquery, RestAPI

in order to make a search, user must enter name of city, and name of state. 
Using MapQuest to get Longtitude and Latitude. Then, using DarkSky to get weather in details.

The project does  not have a function to save a location. So, whenever user exits website, all information will be eareased
